# Contributing to AI-102 Study Resources

Thank you for your interest in contributing! This repository contains study materials for the Microsoft AI-102 Azure AI Engineer Associate certification exam.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test any code samples
5. Commit your changes (`git commit -m 'Add: your feature description'`)
6. Push to the branch (`git push origin feature/your-feature`)
7. Open a Pull Request

## Guidelines

- Keep code samples clear and well-commented
- Include Python and REST API examples where possible
- Reference official Microsoft documentation
- Follow the existing file structure and naming conventions
- Test all Bicep templates before submitting
