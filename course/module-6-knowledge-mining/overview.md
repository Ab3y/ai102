# Module 6: Implement Knowledge Mining and Document Intelligence Solutions

## Exam Weight: 15–20%

---

## Module Overview

This module covers Azure AI Search for knowledge mining, Azure AI Document Intelligence for structured document processing, and Azure AI Content Understanding for complex content extraction. You will learn to build search indexes, implement AI enrichment pipelines, extract data from documents, and process multimedia content.

---

## Learning Objectives

After completing this module, you will be able to:

- Provision and configure Azure AI Search
- Create search indexes with appropriate field attributes
- Configure indexers and data sources
- Implement built-in and custom skillsets for AI enrichment
- Create knowledge store projections
- Write search queries using OData and simple syntax
- Implement semantic and vector search
- Analyze documents with prebuilt and custom Document Intelligence models
- Build composed models for complex document scenarios
- Extract text, tables, and entities from complex documents with Content Understanding
- Process video and audio content with Content Understanding
