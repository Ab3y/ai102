# Module 5: Implement Natural Language Processing Solutions

## Exam Weight: 15–20%

---

## Module Overview

This module covers Azure AI Language and Speech services for processing, analyzing, and generating natural language. You will learn text analytics, speech processing, translation, and how to build custom language models for domain-specific understanding.

---

## Learning Objectives

After completing this module, you will be able to:

- Analyze text for sentiment, key phrases, entities, and PII
- Detect language and link entities to knowledge bases
- Translate text and documents between languages
- Implement speech-to-text and text-to-speech conversion
- Write and customize SSML for speech synthesis
- Build custom speech models
- Create Conversational Language Understanding (CLU) models
- Implement custom question answering solutions
- Configure speech translation
