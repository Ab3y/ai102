# Module 2: Implement Generative AI Solutions

## Exam Weight: 15–20%

This module covers Azure OpenAI, Microsoft Foundry, and the operational aspects of generative AI. Expect 9–12 questions on these topics.

---

## Learning Objectives

After completing this module, you will be able to:

- Build solutions using Microsoft Foundry (hubs, projects, prompt flow, RAG)
- Provision and use Azure OpenAI Service (chat completions, embeddings, DALL-E)
- Optimize generative AI with parameter tuning, prompt engineering, and fine-tuning
- Operationalize generative AI solutions for production

## Prerequisites

- Completed Module 1 (Plan and Manage Azure AI Solutions)
- Azure subscription with Azure OpenAI access approved
- Familiarity with REST APIs and Python
- Basic understanding of machine learning concepts

## What This Module Covers

| Area | Description |
|------|-------------|
| **Microsoft Foundry** | Hub/project architecture, model catalog, prompt flow, RAG pattern, evaluation |
| **Azure OpenAI** | Model deployment, chat completions API, parameters, DALL-E, embeddings |
| **Optimization** | Prompt engineering, fine-tuning, parameter tuning, model monitoring |
| **Operations** | Tracing, feedback, container deployment, multi-model orchestration |
