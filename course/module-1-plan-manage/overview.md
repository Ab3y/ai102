# Module 1: Plan and Manage Azure AI Solutions

## Exam Weight: 20–25%

This is the **highest-weighted** domain on the AI-102 exam. Expect 12–15 questions covering service selection, provisioning, security, monitoring, and responsible AI practices.

---

## Learning Objectives

After completing this module, you will be able to:

- Select the appropriate Azure AI service for a given scenario
- Plan, create, and deploy Azure AI service resources
- Manage, monitor, and secure Azure AI services
- Implement responsible AI practices including content safety

## Prerequisites

- Azure subscription with contributor access
- Basic understanding of Azure portal, CLI, and ARM templates
- Familiarity with REST APIs and at least one SDK language (Python or C#)
- Understanding of Azure networking and identity concepts

## What This Module Covers

| Area | Description |
|------|-------------|
| **Service Selection** | Choosing the right AI service (Cognitive Services, OpenAI, AI Search, Document Intelligence, etc.) |
| **Provisioning** | Creating resources via Portal, CLI, Bicep; SDK setup; container deployment |
| **Security** | Key management, managed identity, RBAC, Key Vault, Private Link |
| **Monitoring** | Azure Monitor, Log Analytics, diagnostic settings, alerts, cost management |
| **Responsible AI** | Content Safety, content filters, prompt shields, blocklists, governance |
