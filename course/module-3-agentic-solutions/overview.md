# Module 3: Implement Agentic AI Solutions

## Exam Weight: 5–10%

This is the newest domain on the AI-102 exam. While it has the lowest weight (3–6 questions), it covers emerging concepts around AI agents that are increasingly important in production scenarios.

---

## Learning Objectives

After completing this module, you will be able to:

- Understand AI agent concepts (tools, memory, orchestration, planning)
- Build custom agents using Microsoft Foundry Agent Service
- Configure agent tools: code interpreter, file search, and function calling
- Design multi-agent orchestration patterns
- Implement safety guardrails and testing for agents
- Deploy and optimize agents for production

## Prerequisites

- Completed Module 1 (Plan and Manage) and Module 2 (Generative AI)
- Understanding of Azure OpenAI chat completions API
- Familiarity with function calling concepts
- Azure subscription with Foundry access

## What This Module Covers

| Area | Description |
|------|-------------|
| **Agent Concepts** | Tools, memory, orchestration, planning, autonomy levels |
| **Foundry Agent Service** | Creating agents, configuring tools, deploying |
| **Multi-Agent Systems** | Planner-executor patterns, agent orchestration |
| **Safety & Operations** | Guardrails, testing, monitoring, deployment |
